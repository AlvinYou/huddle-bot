# huddle-bot

Slack 데일리 허들 출석 현황을 자동으로 체크하는 봇.

평일 09:15에 허들이 시작되면 스레드에 팀별 참여 현황을 올리고, 참여자가 바뀔 때마다 메시지를 업데이트한다. 전원 참여 시 조기 종료.

```
📋 오늘 데일리 허들 출석 현황

✅ 이사진 3/3
✅ PMO팀 2/2
⏳ UX/UI팀 1/2  (미참여: 홍길동)
✅ 인프라서비스개발팀 3/3
✅ 앱서비스개발팀 3/3
⏳ 웹서비스개발팀 2/3  (미참여: 김철수)
✅ AI CORE팀 2/2
```

## 연차/부재 연동 (선택)

카카오웍스 캘린더의 iCal 공개 링크(`KAKAOWORK_ICAL_URL`)를 설정하면, 그날의 연차·반차·부재 일정을 출석 현황에 함께 반영한다. 설정하지 않으면 연동 없이 기존 출석 체크만 동작한다.

- **오전을 비우는 부재**(연차, 오전 반차, 출산휴가, 예비군 등)는 09:15 허들 미참여가 정상이므로 출석 분모에서 제외한다.
- **오후만 비우는 부재**(오후 반차 등)는 오전 허들엔 참석해야 하므로 출석 계산엔 정상 포함한다.
- 오전·오후 부재자는 메시지 맨 아래 **"오늘의 부재"** 섹션에 팀/이름/사유로 한 번에 모아 브리핑한다.
- 캘린더 이벤트는 `SUMMARY`의 이름으로 Slack 멤버와 매칭한다.

```
📋 오늘 데일리 허들 출석 현황

✅ PMO팀 3/3
✅ 웹서비스개발팀 5/5
⏳ AI CORE팀 1/2  (미참여: 김철수)

🟢 출근 인원 전원 참여! 시작하세요!

────────────
🌴 오늘의 부재
· PMO팀 홍길동 (출산휴가)
· 웹서비스개발팀 이몽룡 (오후 반차)
```

## 기술 스택

- Node.js 20 / TypeScript
- AWS Lambda + EventBridge (AWS SAM)
- Slack Web API (`@slack/web-api`)
